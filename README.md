# Educational Course Portal

## Overview

The Educational Course Portal is a simple web-based application developed using HTML5 and Internal CSS. It provides an organized interface for learners to explore featured courses, view course details, access learning resources, and contact the support team. The project demonstrates the use of semantic HTML elements, hyperlinks, tables, lists, images, and basic CSS styling.

## Features

* Responsive page layout using HTML5 semantic elements
* Internal navigation using bookmark links
* External resource link
* Featured courses section with clickable banner image
* Course catalog displayed in a table
* Nested ordered and unordered lists for course modules and resources
* Description list for additional course information
* About and Contact sections
* Sidebar with quick navigation links
* Email and telephone hyperlinks
* Internal CSS for styling and layout

## Technologies Used

* HTML5
* Internal CSS

## How to Run

1. Download or clone the project.
2. Open the project folder.
3. Double-click **index.html** or open it in any modern web browser.
4. Explore the different sections using the navigation menu.
